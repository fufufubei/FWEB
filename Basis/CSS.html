    # ios文字两端对齐兼容

    在ios，两端对齐无效

    p{
    text-align:justify;
}

    兼容写法：

    p {                 
     height: 32/@rem;                   
     text-align: justify;
     &:after {
         display: inline-block;
         width: 100%;
         content: '';
     }
}
